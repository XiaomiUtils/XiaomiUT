# XiaomiUT 🚀

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Status: WIP](https://img.shields.io/badge/Status-Work%20In%20Progress-orange)

**XiaomiUT** 是一个帮助用户从官方镜像站直接查找并下载小米官方固件的网站。无广告，纯净体验。

> [!WARNING]  
> **本项目目前处于开发阶段（WIP）。**  
> 功能可能不稳定、不完整，或随时发生重大变更。请谨慎使用。

---

## ✨ 主要功能（已规划 & 开发中）

* **智能搜索：** 通过设备代号快速查找固件
* **直链下载：** 直达小米官方镜像（bigota / hugeota）下载链接
* **区域支持：** 中国大陆、国际版、欧洲经济区（EEA）、印度版等
* **刷机方式：** 支持 OTA 卡刷包与 Fastboot 线刷包

## 🛠 技术栈

* **前端框架：** [React 18](https://reactjs.org/)（基于 Vite）
* **开发语言：** [TypeScript](https://www.typescriptlang.org/)
* **UI 组件库：** [Ant Design (antd)](https://ant.design/)
* **图标库：** [@ant-design/icons](https://ant-design.github.io/icons/)
* **网络请求：** [Axios](https://axios-http.com/)（用于 API 调用）
